# みんなで逮捕されようプロジェクト

日本の番号法では、特定個人情報の提供が制限されています。

e-Gov 法令検索に掲載されている「行政手続における特定の個人を識別するための番号の利用等に関する法律」第十九条は、例外に当たる場合を除き、何人も特定個人情報を提供してはならないと定めています。

同法第四十八条は、個人番号利用事務等に従事する者又は従事していた者が、正当な理由なく、その業務で取り扱った個人の秘密に属する事項が記録された特定個人情報ファイルを提供した場合に、四年以下の拘禁刑若しくは二百万円以下の罰金又は併科を定めています。同法第四十九条は、同じ立場の者が、業務上知り得た個人番号を不正な利益を図る目的で提供し、又は盗用した場合に、三年以下の拘禁刑若しくは百五十万円以下の罰金又は併科を定めています。

よってこれは日本では「犯罪」と見做されます。それではみんなで犯罪者になり逮捕されましょう！

良い犯罪者生活を！

## 逮捕される方法

このリポジトリを公開し、`index.html` へのリンクを共有してください。

GitHub Pages などで公開し、URL を共有してください。公開先で `index.html` が開ければ動きます。

## なかなか逮捕されませんか？

警察に自首することができます。
